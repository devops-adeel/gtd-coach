GTD COACH QUICK REFERENCE
========================
Now with automatic memory tracking, pattern detection, Timing integration & LLM monitoring!

TIMING SETUP (One-Time)
Copy .env.example to .env, add API key from web.timingapp.com

START YOUR REVIEW (Native Python)
$ ~/gtd-coach/start-coach.sh

START YOUR REVIEW (Docker - Recommended)
$ ./docker-run.sh

REVIEW PHASES (30 minutes)
--------------------------
1. STARTUP     (2 min)  - Get ready, coach welcomes you
2. MIND SWEEP  (10 min) - Type everything on your mind
3. PROJECTS    (12 min) - One next action per project (45 sec each)
4. PRIORITIES  (5 min)  - Pick top 5-7 items, assign A/B/C
5. WRAP-UP     (3 min)  - Review stats and celebrate!

KEY INTERACTIONS
----------------
- Press ENTER after each item/response
- Keep responses short
- Trust the time limits
- First thought = best thought
- Ctrl+C to emergency stop (saves progress)

AUDIO ALERTS
------------
- 50% time: Tink sound
- 20% time: Tink sound  
- 10% time: Purr sound
- Complete: Glass sound

AFTER YOUR REVIEW
-----------------
Your data saved in:
- ~/gtd-coach/data/mindsweep_[date].json
- ~/gtd-coach/data/priorities_[date].json
- ~/gtd-coach/logs/review_[date].json
- ~/gtd-coach/data/graphiti_batch_[date].json (memory tracking)

View weekly insights:
$ python3 ~/gtd-coach/generate_summary.py

DOCKER COMMANDS (If using Docker)
---------------------------------
Build image:    ./docker-run.sh build
Run review:     ./docker-run.sh
Test Timing:    ./docker-run.sh timing
Test Langfuse:  ./docker-run.sh test
Weekly summary: ./docker-run.sh summary

TEST COMMANDS (Native Python)
-----------------------------
Timer test:     ~/gtd-coach/scripts/timer.sh 1 "Test done!"
Connection:     python3 ~/gtd-coach/test-simple-prompt.py
Demo review:    python3 ~/gtd-coach/demo-review.py
Langfuse:       python3 ~/gtd-coach/test_langfuse.py

LANGFUSE SETUP (Optional)
-------------------------
1. Copy config: cp langfuse_tracker.py.example langfuse_tracker.py
2. Add your keys to langfuse_tracker.py
3. Test: ./docker-run.sh test (or python3 test_langfuse.py)
4. View metrics at http://localhost:3000
